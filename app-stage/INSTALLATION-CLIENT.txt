========================================
    GESTION ADMINISTRATIVE
    Instructions d'installation
========================================

ÉTAPE 1 : INSTALLER NODE.JS
- Téléchargez et installez Node.js depuis https://nodejs.org/
- Choisissez la version LTS (recommandée)
- Vérifiez l'installation en ouvrant un terminal : node --version

ÉTAPE 2 : RESTAURER L'APPLICATION (SI NÉCESSAIRE)
- Si l'application ne démarre pas, double-cliquez sur "RESTORE-APP.bat"
- Ce script réinstalle automatiquement toutes les dépendances
- Attendez que la restauration soit terminée

ÉTAPE 3 : DÉMARRER L'APPLICATION
- Double-cliquez sur "start-complete-app.bat" (lanceur principal)
- L'application se lance automatiquement sur le port 3000
- Laissez les fenêtres ouvertes pendant l'utilisation

ÉTAPE 4 : UTILISATION
- L'application s'ouvre dans une fenêtre Electron
- Toutes les fonctionnalités sont accessibles depuis le menu de gauche
- L'interface s'adapte automatiquement à votre écran
- L'application fonctionne en mode développement (electron-dev)

ÉTAPE 5 : ARRÊT
- Fermez la fenêtre de l'application pour l'arrêter
- Fermez les fenêtres de terminal pour arrêter les services

========================================
    CONFIGURATION PORTS
========================================

L'application est configurée pour fonctionner sur le port 3000 :
- Serveur backend : http://localhost:3000
- Application frontend : http://localhost:3000
- Mode : Développement (electron-dev)

========================================
    EN CAS DE PROBLÈME
========================================

Si l'application ne démarre pas :

1. **Vérifiez que Node.js est bien installé** et redémarré
2. **Utilisez "RESTORE-APP.bat"** pour réinstaller les dépendances
3. **Essayez le lanceur principal** "start-complete-app.bat"
4. **Vérifiez que le port 3000 est libre** (pas d'autre application)
5. **Vérifiez votre connexion internet** (pour télécharger les dépendances)
6. **Redémarrez votre ordinateur**
7. **Contactez le support technique**
